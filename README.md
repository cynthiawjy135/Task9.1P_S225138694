## Task 9.1P Lost and Found Items App with Google Maps API
In this app, user can add ew advert and upload the picture of the Items and using PlacesAutocomplete from Google for the location to easily input the specific location. User also can get all the lost or found Items nearby their location in Maps view in 5km distance.

The components that are used in this app are:
- RoomDB
- RecyclerView
- Google Maps API
- Image Upload

Requirements to run this app
- Android Studio Panda
- Emulator
- minSDK = 24
